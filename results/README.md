# CNBE-32: Historical Experimental White Papers

This directory contains centralized copies of historical white papers from across the repository, organized by experiment phase.

These documents are archived exploratory artifacts.

They are kept for provenance and should not be read as current confirmatory evidence, production-readiness claims, financial advice, hardware-readiness claims, or universal CNBE effectiveness claims.

For current protocol boundaries, use:

- `llm_experiments/v1_v4_redesign/`
- `llm_experiments/v5_v10_redesign/`
