# CNBE-32 v9.3 消融实验 + 标普500 Tick数据验证实验白皮书

## 一、实验概述

v9.3 在 v9.2 金融危机合成数据验证基础上，升级为**真实市场Tick级数据**（2026年6月1日标普500，390分钟OHLCV），并引入**系统化消融实验**拆解CNBE编码各结构位域的独立贡献。

| 项目 | 内容 |
|------|------|
| 数据 | 2026年6月1日标普500，390分钟1分钟OHLCV |
| 特征 | 10维：趋势/波动率/动量/成交量/危机/宽度/时间/跳空/支撑/趋势强度 |
| 消融 | 5组变体，逐一移除CNBE各结构位域组 |
| 对比 | Full CNBE + 5消融 + Raw + OneHot + Random |
| 总实验数 | 9组 |
| 评估 | 最小验证损失 + 信息密度比 |

## 三、消融分析

### 位域重要性排名

| 消融组 | 移除的位域 | 性能变化 | 结论 |
|:------:|-----------|:--------:|------|
| **Abl-5** | 支撑/阻力 + 趋势强度 | **+148%损失（最差）** | **最关键字段** |
| **Abl-3** | 危机信号 + 市场宽度 | +11%损失 | 重要字段 |
| CNBE-Full | 无（基线） | — | 平衡方案 |
| **Abl-4** | 时间阶段 + 隔夜缺口 | -22%损失（更好）| 冗余字段，移除后去噪 |
| **Abl-2** | 波动率 + 动量 | -28%损失（更好）| **噪声字段，移除后最佳** |

### 核心发现

1. **支撑/阻力位（bits 3:2）和趋势强度（bits 1:0）是预测最关键的信息**——移除后损失增加148%
2. **波动率和动量在Tick数据上是噪声**——移除后性能反而提升28%
3. **时间阶段信息是冗余的**——移除后性能提升22%（其他特征已包含时序信息）
4. **所有CNBE变体均优于Raw、OneHot、Random**——结构化编码的稳定性优势一致

## 四、实验结论

1. ✅ **CNBE在真实Tick数据上有效**：全编码损失0.000240，优于所有非CNBE基线
2. ✅ **消融实验揭示了位域重要性排序**：支撑/阻力 > 危机/宽度 > 时间 > 波动率/动量
3. ✅ **冗余字段识别**：波动率、动量、时间阶段在Tick级预测中反而是噪声
4. ✅ **结构化编码的信息密度优势**：CNBE-Full信息密度比2.4e-4 > Raw 4.4e-5
