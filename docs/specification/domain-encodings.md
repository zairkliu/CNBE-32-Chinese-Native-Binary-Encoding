## 位域设计原则

1. **总宽 32 位**: 对齐 RISC-V 通用寄存器
2. **语义对齐**: 每字段有独立物理意义
3. **分辨率适配**: 量化精度匹配观测精度
4. **扩展预留**: 高位保留给未来扩展
