# 32位位域布局（核心规范）

## 规范版本

CNBE-32 有两种位域布局规范:
1. **汉字规范**（v6.0 定稿）：用于 CJK 字符编码
2. **通用规范**（v9.0 起用）：用于跨领域实验

---

## 一、汉字规范（CJK 字符）

```
[31:24] 部首区 (8bit) - 214康熙部首 + 扩展
[23:19] 笔画区 (5bit) - 1-31画
[18:15] 结构区 (4bit) - 9种结构类型
[14:4]  字库区 (11bit) - 组内索引（20902 CJK字）
[3:0]   扩展区 (4bit) - 繁简/多音/异体标志
```

### 字段详解

| 字段 | 位域 | 范围 | 说明 |
|------|:----:|:----:|------|
| 部首 | [31:24] | 0-255 | 214康熙部首 + 41现代扩展 |
| 笔画 | [23:19] | 0-31 | 实际笔画数 |
| 结构 | [18:15] | 0-15 | 0=独体,1=左右,2=左中右,3=上下,4=上中下,5=左上包围,6=右上包围,7=左下包围,8=上包围,9=下包围,10=左包围,11=全包围,12=品字 |
| 字库 | [14:4] | 0-20902 | Unicode 0x4E00-0x9FA5 内的索引 |
| 扩展 | [3:0] | 0-15 | bit0:繁简, bit1:古今, bit2:方言, bit3:保留 |

### 编码示例

| 汉字 | CNBE (hex) | 部首 | 笔画 | 结构 |
|------|:----------:|:----:|:----:|:----:|
| 中 | 0x274C6010 | 39 | 4 | 0(独体) |
| 国 | 0x04808001 | 31 | 8 | 11(全包围) |
| 明 | 0x0072C008 | 72 | 8 | 1(左右) |
| 语 | 0x00944009 | 149 | 9 | 1(左右) |

### C 结构体

```c
typedef struct {
    uint32_t radix     : 8;  // 位 24-31
    uint32_t stroke    : 5;  // 位 19-23
    uint32_t structure : 4;  // 位 15-18
    uint32_t index     : 11; // 位 4-14
    uint32_t extension : 4;  // 位 0-3
} __attribute__((packed)) cnbe32_cjk_t;
```

### 查询函数

```c
uint32_t cnhe_map(uint32_t unicode);   // Unicode → CNBE
uint32_t cnhe_extract(uint32_t code, uint32_t field);  // 位域提取
uint32_t cnhe_cmp(uint32_t a, uint32_t b);  // 汉明距离
```

## 二、通用规范（跨领域实验）

```
[31:28] 符号类型 (4bit) - 0=数字,1=变量,2=运算符,3=关系,4=函数,5=括号
[27:24] 符号子类 (4bit) - 类型内具体分类
[23:20] 优先级 (4bit) - 运算优先级
[19:16] 参数数量 (4bit) - 0/1/2/3元操作
[15:12] 数值量级 (4bit) - 数值的log2范围
[11:8]  关系类型 (4bit) - =,<,>,<=,>=,!=等
[7:4]   复杂等级 (4bit) - 嵌套/深度等级
[3:0]   方向标志 (4bit) - 左/右结合,对称等
```

### 实验编码方案

各实验使用不同的位域分配（见 [domain-encodings.md](domain-encodings.md)）:

| 实验 | 编码字段 | 位数 |
|:----:|----------|:----:|
| v9.0 树木 | species(4)+height(6)+crown(6)+health(4)+light(4)+water(4)+age(4) | 32 |
| v10.3 台风 | lat(8)+lon(8)+wind(8)+pressure(8) | 32 |
| v10.5 黑洞 | r_schwarzschild(8)+redshift(8)+tidal(8)+deflection(8) | 32 |
| v10.6 城市 | traffic(8)+livelihood(6)+infra(6)+env(4)+emergency(3)+region(3)+time(2) | 32 |
| v10.7-8 数学 | token_type(4)+value(4) | 8(有效) |
