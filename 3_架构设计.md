# CNBE-32 全链路架构设计

**版本**: 2.0 | **日期**: 2026-07-02  

---

## 一、六层架构

```
┌─────────────────────────────────────────┐
│  第6层: 应用层                           │
│  XLSX/SQLite/CSV 数据库 / 搜索引擎       │
├─────────────────────────────────────────┤
│  第5层: 知识库层                          │
│  GB 18030-2022 / GB/T 22320-2025 对接    │
├─────────────────────────────────────────┤
│  第4层: AI 层                            │
│  部首特征提取 / 聚类验证 / 距离度量       │
├─────────────────────────────────────────┤
│  第3层: 系统层                            │
│  哈希索引 (15bit×4) / 二分查找 (O(log n))│
├─────────────────────────────────────────┤
│  第2层: 芯片层                            │
│  RISC-V Spike / 6 条自定义指令 (Custom-0)│
├─────────────────────────────────────────┤
│  第1层: 编码层                            │
│  CNBE-32 / Unicode 互转 / 三进制校验      │
└─────────────────────────────────────────┘
```

## 二、数据流

```
输入 (Unicode 字符串)
  ↓
Unicode → CNBE-32 查找 (哈希 O(1) / 二分 O(log n))
  ↓
CNBE-32 编码序列 (32-bit 定长数组)
  ↓
CPU 指令处理 (cnbe.rad/str/struct/dist)
  ↓
结果输出 (部首/笔画/结构/距离)
  ↓
数据库写入 (XLSX / SQLite / JSON / CSV)
```

## 三、存储架构

| 格式 | 用途 | 大小 |
|:----:|:----:|:----:|
| XLSX | 可视化查阅 | 5.7 MB |
| SQLite | 程序查询 | 5.1 MB |
| JSON | 跨语言交换 | 11 MB |
| CSV | 通用数据 | 4.7 MB |
| C Header | Spike 集成 | 3.6 MB |
