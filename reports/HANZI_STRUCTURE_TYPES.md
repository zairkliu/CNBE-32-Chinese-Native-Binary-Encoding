# Hanzi Structure Types

Only these structure values are allowed for normalized Hanzi structure review.

## Special Case

- `独体字`: 结构上不拆分为两个及以上部件。

## Twelve Allowed Structures

| ID | Structure | Description |
| --- | --- | --- |
| 1 | 上下 | 上部和下部组合。 |
| 2 | 上中下 | 上、中、下三段组合。 |
| 3 | 左右 | 左部和右部组合。 |
| 4 | 左中右 | 左、中、右三段组合。 |
| 5 | 左上包 | 左上方向包围或覆盖内部部件。 |
| 6 | 右上包 | 右上方向包围或覆盖内部部件。 |
| 7 | 左三包 | 左侧三面包围。 |
| 8 | 左下包 | 左下方向包围或承托内部部件。 |
| 9 | 上三包 | 上方三面包围。 |
| 10 | 下三包 | 下方三面包围。 |
| 11 | 全包围 | 外部部件完整包围内部部件。 |
| 12 | 镶嵌 | 部件交错嵌合，不能简单归入普通上下或左右结构。 |

## Forbidden Structure Labels

- `品字形`
- `三叠结构`
- `三金结构`
- `会意结构`
- Any other label not listed above.

Such descriptions may appear only in semantic or etymology notes, never in the normalized structure field.
Any new label must first be traced to a cited national standard before it can enter the controlled vocabulary.
